# Dumb Security

A playground for some OAuth2/OIDC/JWT stuff

## Endpoints
* /hello
* /authorizedHello
* /api/oldauth
* /api/auth/refresh
* /api/auth/signup
* /api/auth/signin

* /api/test/user for users has ROLE_USER or ROLE_MODERATOR or ROLE_ADMIN
* /api/test/mod for users has ROLE_MODERATOR
* /api/test/admin for users has ROLE_ADMIN
